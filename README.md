# Diffusion-Policy-PushT
Simple-to-use PushT task-based Diffusion Policy

This project is adapted from the source code of Diffusion Policy ( https://github.com/real-stanford/diffusion_policy ). To train and evaluate the model, first download the data from https://diffusion-policy.cs.columbia.edu/data/training/pusht.zip and place it in the data folder in the root directory.
